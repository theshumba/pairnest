# 🪐 PAIRNEST: GALAXY EDITION

> **Status:** SYSTEMS ONLINE
> **Version:** 2.0 (PLASTIC UPDATE)
> **Aesthetic:** High-Gloss Y2K / Social RPG

## 🚀 MISSION BRIEFING
We have abandoned the "Sad Beige" aesthetic of the old internet. We are building a digital playground for couples that looks like a Game Boy Advance exploded inside a glitter factory.

## 🛠️ TECH STACK
- **Engine:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS + Custom "Plastic" Tokens
- **Physics:** Framer Motion
- **State:** Zustand

## 🎨 DESIGN SYSTEM: "PLASTIC UI"
The UI is built on the core principle of **"Juice"**.
- **Buttons:** Must act like physical objects. Press them.
- **Panels:** Frosted glass with holographic borders.
- **Text:** Chunky `Fredoka` for headers, `VT323` for data.

## 🎮 HOW TO START
1. `npm install`
2. `npm run dev`
3. Open `localhost:3000`
